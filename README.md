# Woodpecker Method Chess Training ♟️
A fast, offline-capable Progressive Web App (PWA) designed to help chess players train tactical patterns using the **Woodpecker Method** (spaced repetition).
🔗 **[Play the App Live Here](https://saixua.github.io/woodpecker-chess/)**
## 📖 What is the Woodpecker Method?
The Woodpecker Method is a chess training system focused on building deep, subconscious pattern recognition. Instead of solving thousands of different puzzles once, you solve a curated set of puzzles repeatedly.
1. **Cycle 1**: Solve a set of puzzles. Take as much time as you need to get them right.
2. **Cycle 2**: Solve the exact same set of puzzles again, aiming to finish in **half the time**.
3. **Subsequent Cycles**: Repeat the process up to 7 times. Eventually, you will solve the entire set almost instantly, proving the patterns are hardwired into your brain!
## ✨ Features
- **Massive Database**: 14,000+ curated open-source puzzles from the Lichess database.
- **Custom Drills**: Filter puzzles by specific tactical motifs (e.g., Smothered Mate, Forks, Pins) and Elo difficulty (Beginner to Master).
- **Offline Capable**: Installed as a PWA, you can train on an airplane without an internet connection.
- **Advanced Tracking**: Automatically tracks your cycle history, time spent, and failed puzzles.
- **Drill Failed Puzzles**: Instantly generate a new training cycle containing only the puzzles you missed in your last session.
## 📱 Installation
Because this is a Progressive Web App, you can install it directly to your phone's home screen!
- **iOS (Safari)**: Tap the `Share` button at the bottom of the screen, then tap `Add to Home Screen`.
- **Android (Chrome)**: Tap the three dots `⋮` in the top right, then tap `Install App` or `Add to Home Screen`.
